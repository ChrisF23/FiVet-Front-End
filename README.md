# FiVet Front-End

## Instalacion:
```
npm install
```
## Compilar y ejecutar:
```
npm run serve
```

## Estructura del proyecto:
* **src/components**: Componentes de la pagina. Por ejemplo, la barra de navegacion.
* **src/views**: Contiene las vistas. Son el contenido de la pagina.
* **src/router.js**: Especificacion de las rutas.
